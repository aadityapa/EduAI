# Teacher Onboarding & Training — Pilot

---

## Session 1: Platform Basics (90 min)

1. Login and dashboard overview
2. Class roster and attendance
3. Assigning homework and viewing submissions
4. Using AI question generator (teacher-only)

**Hands-on:** Mark attendance for demo class, create one assignment

---

## Session 2: AI & Assessment (90 min)

1. AI tutor boundaries (students vs teachers)
2. Creating and assigning quizzes
3. Reading analytics and parent reports
4. Escalating content issues

**Hands-on:** Generate 5-question quiz, assign to class

| Task | Path |

## Certification

Teachers complete 10-question checklist quiz (80% pass) before full class rollout.
