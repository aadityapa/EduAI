# Student Onboarding Guide — Pilot

---

## First Login

1. Get credentials from teacher (email + temp password)
2. Open app or `https://[school].eduai.in`
3. Complete profile: name, class, avatar
4. Optional: select Hindi/Marathi interface

---

## Daily Use

| Activity | Where |
|----------|-------|
| Watch lessons | Courses → Subject → Lesson |
| Ask AI tutor | Tutor tab (class-appropriate) |
| Take quiz | Quizzes → Assigned |
| Study plan | Planner (AI-generated weekly plan |
| Offline | Download lessons on WiFi (mobile) |

---

## Rules of Use

- AI tutor helps learn — don't ask for exam answers during live tests
- Report inappropriate AI responses to teacher
- Keep login private; don't share password

---

## Gamification

- Earn XP for lessons and quizzes
- Streaks for daily login
- Leaderboard is class-scoped (optional per school)
