Understanding the General Form

A quadratic function is a polynomial function of degree two. The graph of a quadratic function is a curve called a parabola. In this lesson, we focus on how the coefficients impact the shape and position of the curve.

General Form Equation

f(x) = ax² + bx + c

Vertex Form

f(x) = a(x - h)² + k

Where (h, k) is the vertex of the parabola.

The Discriminant

D = b² - 4ac

Determines the number and type of roots.

lightbulb

Key Takeaways

  • check_circle If a > 0, the parabola opens upwards (minimum).
  • check_circle If a < 0, the parabola opens downwards (maximum).
  • check_circle The y-intercept is always at the point (0, c).
