# App Store Release Checklist

## Pre-Submission

- [ ] Legal review of Privacy Policy and Terms (hosted at public URLs)
- [ ] App icons: 1024×1024 (iOS), adaptive icon (Android)
- [ ] Screenshots for all required device sizes
- [ ] Age/content rating questionnaire completed
- [ ] Export compliance (encryption) declaration
- [ ] Demo account credentials for app review

## Android (Google Play)

- [ ] Google Play Developer account active
- [ ] App bundle (AAB) signed with upload key
- [ ] Data safety form completed
- [ ] Internal testing track → closed testing → production
- [ ] Target API level meets Play requirements

## iOS (App Store Connect)

- [ ] Apple Developer Program enrolled
- [ ] App Store Connect app record created
- [ ] Provisioning profiles and certificates
- [ ] TestFlight beta with pilot schools (min 2 weeks)
- [ ] App Review notes with demo login

## Post-Launch

- [ ] Monitor crash rate (Sentry)
- [ ] Respond to store reviews within 48h
- [ ] Track install/uninstall analytics
