Key Takeaways

Vertex Form

The vertex is always located at point (h, k).

Parabola Direction

If a > 0, the parabola opens upward (U-shape). If a < 0, it opens downward.

Standard Form

f(x) = ax² + bx + c, where c is the y-intercept.
