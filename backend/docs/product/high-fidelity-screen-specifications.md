# High-Fidelity Screen Specifications

This document is the mandatory pre-code screen specification library for EduAI's next-generation product experience. It defines 80 high-fidelity screen specs before frontend implementation.

Each screen must feel AI-first, emotionally engaging, mobile-first, accessible, and premium.

## Student Experience

### 1. Student Home

Purpose: Replace the dashboard with an immersive daily launchpad.

Emotional target: "I want to learn today."

Primary action: Continue today's recommended lesson.

Secondary action: Ask AI Tutor.

Progress: 7-day streak, XP, rank, lessons remaining, daily mission progress.

Personalization: Greeting by name, time-of-day mood, subject recommendation.

Gamification: XP count-up, streak flame, rank movement preview.

AI assistance: "I picked this lesson because you struggled with fractions yesterday."

Mobile: Hero first, then horizontal cards, bottom navigation.
