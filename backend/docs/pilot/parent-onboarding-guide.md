# Parent Onboarding Guide — Pilot

---

## Getting Started

1. Receive invite email or SMS from school
2. Download EduAI app (iOS/Android) or use web portal
3. Login with email + temporary password → set new password
4. Link to child profile(s) — verify with school registration number

---

## Key Features

| Feature | Description |
|---------|-------------|
| Progress | View child's course completion and quiz scores |
| Fees | Pay school fees (if ERP enabled) |
| Notifications | Attendance alerts, homework reminders |
| AI Tutor | Monitor usage; tutor is student-facing only |

---

## Parent Orientation (30 min webinar)

- Platform privacy and child data handling
- How to read progress reports
- Fee payment walkthrough
- Support contacts

---

## FAQ

**Q: Can parents use the AI tutor?**  
A: No — AI tutor is for students. Parents see usage summaries.

**Q: Who sees my child's data?**  
A: School teachers/admins and EduAI (as data processor). See Privacy Policy.
